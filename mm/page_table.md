|    字段     |            描述             |      位数      |
| :---------: | :-------------------------: | :------------: |
|     cr3     |        指向一个 PDPT        | crs 寄存器存储 |
|     PGD     | 指向 PDPT 中 4 个项中的一个 |    位 31~30    |
|     PMD     | 指向页目录中 512 项中的一个 |    位 29~21    |
|     PTE     |  指向页表中 512 项中的一个  |    位 20~12    |
| page offset |       4KB 页中的偏移        |    位 11~0     |
